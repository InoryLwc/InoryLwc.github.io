安装VS-code（Visual Studio Code）的过程

安装VS-code（Visual Studio Code）的过程

  • 安装包（安装选择）
  • 语言修改
  • 主题修改
  • 编码修改

安装包

在微软官网下载

  • Microsoft Visual Studio Code

    这是免费的强大的代码编辑器，强大之处在于支持扩展程序。

语言修改

  1. 下载语言包

    2017版本默认为英语，不带中文语言包，需要在扩展程序里搜索Chinese，并且选择
    Chinese (Simplified) Language Pack for Visual Studio Code安装.



  2. 修改配置文件

    • 使用快捷键Ctrl+Shift+P打开软件内部的命令窗口

    • 输入Configure Language回车，在编辑界面找到locale的行将"en"改成"zh-cn"（大小写具体看扩展的说明），然后重启软件即可。



主题修改

VS code自带几款主题：默认暗主题，还有亮主题，茶色主题。

按照顺序文件-首选项-颜色主题打开如下界面，任意选择就可以，想要其他主题就下载扩展程序

修改编码显示文本

PS：VS Code 2017 默认为UTF-8编码显示，如果显示GBK2312编码的文件的中文就会乱码，暂时没找到其他办法，折中一下就是，将GBK2312转换为UTF-8

  • 在扩展里面搜索GBK并找到安装“ChangeEncode”，重启就可以了，但是保存的话还是UTF-8的编码



    （PS：在右下角可以看到当前使用的编码，可以点一下进入选择界面，里面有很多格式的编码，但是就是改不过来。）
